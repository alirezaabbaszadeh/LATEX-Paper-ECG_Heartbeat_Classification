# Journal A Compliance QA Checklist

Use this template to confirm the manuscript satisfies every Journal A submission requirement.

- [ ] Verify manuscript length, abstract structure, and keyword count follow Journal A limits.
- [ ] Check citation style, reference ordering, and DOI formatting.
- [ ] Ensure required statements (ethics, funding, conflict of interest, data availability) are present and properly worded.
- [ ] Confirm supplementary materials meet the journal's file format and size policies.
- [ ] Review cover letter content for journal-specific requests.
- [ ] Validate adherence to 2026 policy updates regarding code/data sharing and reporting standards.
- [ ] Ensure all checklist documents and forms requested by the journal are attached.
- [ ] Document any outstanding compliance issues for resolution before submission.

## Saving Your Review
1. Mark each item as complete (`[x]`) or leave unchecked with notes.
2. Save the checklist in this directory.
3. Commit the changes on a QA branch.
4. Submit a pull request titled `QA: Journal A Compliance Review` so the editorial record stays current.
