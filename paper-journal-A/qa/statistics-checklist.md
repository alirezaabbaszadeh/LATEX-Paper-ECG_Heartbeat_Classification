# Journal A Statistics QA Checklist

Use this template to audit the statistical analysis before Journal A submission.

- [ ] Confirm the statistical hypotheses correspond to the stated research questions.
- [ ] Verify sample size calculations or power analyses are provided when required.
- [ ] Check that assumptions for each statistical test are discussed and validated.
- [ ] Ensure effect sizes, confidence intervals, and p-values are reported per journal standards.
- [ ] Review handling of missing data, outliers, and multiple comparisons.
- [ ] Confirm figures and tables reference the same statistical results described in the text.
- [ ] Ensure reproducibility by citing the software, version, and scripts used for analysis.
- [ ] Note any additional reporting elements mandated by 2026 statistical transparency guidelines.

## Saving Your Review
1. Update the checklist with `[x]` for completed items and add notes as needed.
2. Save the Markdown file after completing your review.
3. Commit the changes on a dedicated branch.
4. Submit a pull request titled `QA: Journal A Statistics Review` to document the results.
