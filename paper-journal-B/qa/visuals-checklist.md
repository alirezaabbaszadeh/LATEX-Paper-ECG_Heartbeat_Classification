# Journal B Visuals QA Checklist

Use this template to assess visual materials prepared for Journal B.

- [ ] Confirm figure resolution, color mode, and dimensions meet Journal B specifications.
- [ ] Ensure axes labels, legends, and annotations follow the journal's typography rules.
- [ ] Verify color palettes remain accessible and align with Journal B brand guidance.
- [ ] Check captions include definitions for abbreviations and describe statistical indicators.
- [ ] Ensure tables provide units, precision, and footnotes per journal requirements.
- [ ] Validate supplemental visuals are referenced correctly in the main text.
- [ ] Confirm embargoed or identifying information is removed from visuals as required.
- [ ] Note compliance with limits on the number of figures/tables.

## Saving Your Review
1. Toggle relevant checkboxes to `[x]` once validated and add issue notes below bullets.
2. Save the file when finished.
3. Commit the update on a dedicated QA branch.
4. Open a pull request titled `QA: Journal B Visuals Review` so the change is recorded.
