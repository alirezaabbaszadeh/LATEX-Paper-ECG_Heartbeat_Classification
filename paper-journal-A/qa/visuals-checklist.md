# Journal A Visuals QA Checklist

Use this template to review figures and tables prepared for Journal A.

- [ ] Confirm figure resolution, dimensions, and file formats comply with the journal guide.
- [ ] Ensure all axes, legends, and annotations are legible and consistent.
- [ ] Verify color choices meet accessibility standards (e.g., color-blind safe palettes).
- [ ] Check that figure captions fully describe the content and abbreviations.
- [ ] Ensure tables include units, statistics, and significant figures required by the journal.
- [ ] Validate that referenced supplemental visuals are provided and labeled correctly.
- [ ] Confirm data availability statements accompany visuals when mandated.
- [ ] Note any figure count or placement issues relative to journal limits.

## Saving Your Review
1. Mark completed checks with `[x]` and document issues directly under each bullet.
2. Save your updates to this file.
3. Commit the changes on a review branch.
4. Open a pull request titled `QA: Journal A Visuals Review` so the team can log the outcome.
