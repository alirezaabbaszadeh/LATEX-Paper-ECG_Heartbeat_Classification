# Changelog

All notable updates to this manuscript will be captured in this file.

## Update Instructions
- Record each release or milestone using the format `## [tag-name] - YYYY-MM-DD`.
- Organize notes under the `### Added`, `### Changed`, and `### Fixed` subsections. Omit a subsection when it would be empty.
- Keep entries focused on reader-facing outcomes (major content shifts, layout tweaks, or fixes) and link to supporting documents or pull requests when available.

Sample entry to illustrate the preferred structure:

## [v0.1-outline] - 2024-05-01
### Added
- Drafted the high-level outline covering objectives, dataset description, and planned figures.

### Changed
- Refined the introduction narrative to foreground the classification problem.

### Fixed
- Corrected terminology inconsistencies between the abstract and outline.
