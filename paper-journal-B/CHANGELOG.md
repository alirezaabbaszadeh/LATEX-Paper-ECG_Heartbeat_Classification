# Changelog

All notable updates to this manuscript submission package will be captured in this file.

## Update Instructions
- Record each release or milestone using the format `## [tag-name] - YYYY-MM-DD`.
- Organize notes under the `### Added`, `### Changed`, and `### Fixed` subsections. Omit a subsection when it would be empty.
- Emphasize publication-ready deliverables (new experiments, layout adjustments, resolved feedback) and link to supporting pull requests when available.

Sample entry to illustrate the preferred structure:

## [v0.1-outline] - 2024-05-01
### Added
- Journal B-specific outline emphasizing clinical validation and deployment considerations.

### Changed
- Reordered results narrative to match Journal B's emphasis on comparative baselines.

### Fixed
- Corrected mislabeled heart rhythm categories in the supplementary materials directory.
