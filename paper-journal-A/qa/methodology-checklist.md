# Journal A Methodology QA Checklist

Use this template to review the methodology section of the Journal A manuscript before submission.

- [ ] Confirm the study objectives align with the journal's stated scope.
- [ ] Verify the experimental design follows the Guide for Authors (e.g., randomized, blinded, prospective).
- [ ] Check that inclusion and exclusion criteria are explicitly defined.
- [ ] Ensure data acquisition hardware, software, and calibration steps are described in sufficient detail for replication.
- [ ] Confirm ethical approvals and informed consent procedures are documented and match journal requirements.
- [ ] Validate preprocessing pipelines (filters, normalization, segmentation) are justified and reproducible.
- [ ] Ensure the methodology references any shared code or datasets as required by 2026 policies.
- [ ] Note deviations from journal formatting limits (word counts, section order) for follow-up.

## Saving Your Review
1. Fill the checklist, converting the relevant checkboxes to `[x]`.
2. Save the updated Markdown file in-place.
3. Commit the changes on a feature branch.
4. Open a pull request titled `QA: Journal A Methodology Review` so the editorial team can track the update.
