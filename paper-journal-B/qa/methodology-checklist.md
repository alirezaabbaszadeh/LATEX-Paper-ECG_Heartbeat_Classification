# Journal B Methodology QA Checklist

Use this template to review the methodology section of the Journal B manuscript before submission.

- [ ] Confirm the study aims align with Journal B's thematic focus.
- [ ] Verify the study design satisfies the journal's methodological expectations (e.g., cohort, cross-sectional, randomized).
- [ ] Check that participant selection criteria and recruitment pathways are documented.
- [ ] Ensure instrumentation, sensor calibration, and signal processing steps are described for reproducibility.
- [ ] Confirm ethics approvals, consent statements, and trial registrations meet Journal B policies.
- [ ] Validate preprocessing, feature engineering, and model training details are transparent.
- [ ] Ensure data/code sharing commitments reflect Journal B's 2026 policy updates.
- [ ] Record any deviations from mandated section ordering or word limits.

## Saving Your Review
1. Mark progress by switching checkboxes to `[x]` and add notes under each bullet if needed.
2. Save the updated Markdown file.
3. Commit the changes on a feature branch dedicated to this review.
4. Open a pull request titled `QA: Journal B Methodology Review` so the editorial board can audit the update.
