# Changelog

All notable updates to this manuscript submission package will be captured in this file.

## Update Instructions
- Record each release or milestone using the format `## [tag-name] - YYYY-MM-DD`.
- Organize notes under the `### Added`, `### Changed`, and `### Fixed` subsections. Omit a subsection when it would be empty.
- Highlight author-facing deliverables such as revised outlines, new figures, or resolved reviewer notes, and link to supporting pull requests when available.

Sample entry to illustrate the preferred structure:

## [v0.1-outline] - 2024-05-01
### Added
- Initial outline tailored to Journal A's section order and word limits.

### Changed
- Updated methods overview to emphasize signal preprocessing choices preferred by Journal A.

### Fixed
- Resolved inconsistencies between the figure captions and text references identified during QA review.
